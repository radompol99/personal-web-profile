# CMSC 207 Mini-Project: Personal Web Profile

This project is a single-page personal web profile built with HTML, CSS, and JavaScript.

## Project Overview

This website presents a student profile with the required CMSC 207 sections and a simple interactive feature.

Current page content includes:
- A header showing `RIO AL-DI A. DOMPOL` and the title **Professional Web Profile**
- A top navigation menu with anchor links to:
  - About Me
  - Hobbies & Interests
  - Contact
- An About Me section with:
  - A profile image
  - A short introduction paragraph
  - A **Show a Fun Fact** button
- A Hobbies & Interests section listing:
  - Reading
  - Gaming
  - Music
- A Contact Info footer with:
  - Email link
  - Social links (Facebook, GitHub)
  - Back-to-top link

## Features

- Semantic HTML structure (`header`, `main`, `section`, `footer`)
- External stylesheet for layout, colors, cards, and hover effects
- JavaScript interactivity via a **Fun Fact** button that cycles through multiple fun facts
- Responsive design using CSS media queries

## File Structure

- `index.html` - Main webpage content and structure
- `styles.css` - Styling and responsive layout
- `script.js` - Fun fact button behavior
- `assets/` - Contains the web assets /js, /img, /css
- `assets/js` - Contains the js files
- `assets/img` - Contains the images
- `assets/css` - Contains the css files


## How to Run

1. Open `index.html` in any web browser.
2. Click **Show a Fun Fact** to test JavaScript interactivity.
3. Resize the browser window to check responsive behavior.
